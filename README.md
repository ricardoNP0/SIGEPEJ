# SIGEPEJ

Sistema de Gestion de Permisos, Justificaciones y Asistencia.

## Estructura

- `backend/`: estructura base para API Node.js, Express, MongoDB y Cloudinary.
- `frontend/`: estructura base para la aplicacion React.
- `docs/`: documentacion tecnica del proyecto.
- `preview/`: prototipo HTML estatico y capturas de referencia.

Las carpetas principales estan vacias con `.gitkeep` para que cada integrante instale y desarrolle su parte en su propia rama.
