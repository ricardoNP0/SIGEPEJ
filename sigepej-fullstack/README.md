# SIGEPEJ - Estructura del proyecto

Estructura base vacia para trabajar por ramas.

Cada integrante debe instalar y configurar su parte en su propia rama:

- `frontend`: React.
- `backend`: Node.js, Express, MongoDB y Cloudinary.
- `docs`: documentacion tecnica, API, base de datos y pantallas.

No hay dependencias instaladas ni archivos generados.
